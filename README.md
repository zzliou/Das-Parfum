* 於專案中使用Pinia狀態管理器，共同管理網頁資料流

* 串接第三方Library如Swiper，並客製化功能與樣式以達成特定功能
* 使用 Git 進行版本操控，並熟稔相關用法
* 使用npm 管理包進行套件控管
* 串接gh-pages服務，並實作客製化指令，讓專案能一鍵部署
